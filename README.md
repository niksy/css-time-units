# css-time-units

[![Build Status][ci-img]][ci]

List of CSS [`<time>`][time] units.

## Install

```sh
npm install css-time-units --save
```

## Usage

```js
var cssTimeUnits = require('css-time-units');
console.log(cssTimeUnits);
/* [
	"s",
	"ms"
] */
```
